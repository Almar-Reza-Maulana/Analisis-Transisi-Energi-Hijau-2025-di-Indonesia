Langkah-Langkah Membuat Conda Environment dengan Anaconda

Halo, teman-teman! Kita akan membangun lingkungan coding menggunakan Anaconda agar proyek energi hijau kita berjalan mulus. Anaconda adalah pengelola Python dan library yang memastikan tidak ada konflik antarproyek. Kita gunakan Python 3.12 untuk stabilitas. Yuk, ikuti langkah-langkah seru ini!

1. Instalasi Anaconda

Narasi: Halo, eco-techno leader! Kita mulai dengan menginstal Anaconda, seperti membangun fondasi rumah coding. Anaconda menyediakan semua alat untuk analisis energi hijau kita.

Langkah:

  • Unduh Anaconda dari https://www.anaconda.com/download untuk Windows/Linux/MacOS.
  • Jalankan installer, ikuti wizard instalasi, dan centang “Add Anaconda to PATH” agar mudah diakses.
  • Buka terminal (Command Prompt di Windows: tekan Win + R, ketik cmd; Mac: Terminal; Linux: Terminal), ketik:
conda --version
  • Jika muncul versi (misalnya conda 24.5.0), instalasi berhasil!

Artinya: Anaconda menjadi pusat kendali, mempersiapkan kita untuk membuat environment proyek.

2. Membuat Conda Environment

Narasi: Halo, teman-teman! Sekarang kita buat ruangan khusus bernama energi_hijau untuk proyek kita. Ini seperti kamar sendiri biar alat coding tidak campur aduk.

Langkah:

  • Di terminal, ketik:
conda create -n energi_hijau python=3.12
  • Tekan y saat diminta konfirmasi.
  • Aktifkan environment:
conda activate energi_hijau
  • Prompt berubah ke (energi_hijau), artinya kita di ruangan baru!

Artinya: Environment memisahkan proyek kita, menggunakan Python 3.12 yang cocok untuk analisis data.

3. Instalasi Library

Narasi: Halo, eco-techno leader! Kita isi ruangan energi_hijau dengan alat-alat keren: Pandas, NumPy, Matplotlib, dan Jupyter. Ini seperti beli perlengkapan untuk analisis canggih.

Langkah:

  • Di terminal (pastikan (energi_hijau) aktif), ketik:
pip install pandas numpy matplotlib jupyter
  • Tunggu instalasi selesai.
  • Cek instalasi di terminal, buka Python (python), ketik:
import pandas
import numpy
import matplotlib
import jupyter
print("Library siap dipake!")
  • Jika tidak ada error, library siap!

Artinya: Library menjadi fondasi untuk mengolah data, menghitung statistik, membuat grafik, dan coding interaktif.

4. Setup VSCode

Narasi: Halo, teman-teman! Kita pindah ke VSCode, seperti membuat meja kerja nyaman untuk coding. VSCode membantu kita menulis dan menjalankan kode dengan rapi untuk analisis energi hijau.

Langkah:

  • Unduh VSCode dari https://code.visualstudio.com.
  • Buka VSCode, klik tab Extensions (Ctrl+Shift+X), instal:
  • Python (by Microsoft): Untuk menjalankan kode Python.
  • Jupyter (by Microsoft): Untuk mendukung file Jupyter Notebook (.ipynb).
  • Klik File > Open Folder, pilih folder C:\EnergiHijau2025 (yang berisi CSV).
  • Buat file Jupyter Notebook: Klik File > New File, pilih Jupyter Notebook, simpan sebagai analisis_energi_hijau.ipynb.

Artinya: VSCode menjadi studio coding, dengan ekstensi Python dan Jupyter memungkinkan kerja interaktif.

Sequence: Setelah environment siap, VSCode menyambungkan ke folder proyek dan data CSV.

5. Mengatur Terminal dan Kernel di VSCode

Narasi: Halo, eco-techno leader! Kita atur terminal dan kernel di VSCode agar kode berjalan di environment energi_hijau. Ini seperti menyambungkan listrik ke meja kerja!

Langkah:

  • Buka terminal baru: Klik Terminal > New Terminal.
  • Pastikan (energi_hijau) aktif. Jika belum, ketik:
conda activate energi_hijau
  • Prompt berubah ke (energi_hijau).
  • Pilih kernel: Di analisis_energi_hijau.ipynb, klik Select Kernel di kanan atas. Pilih energi_hijau (Python 3.12). Jika tidak muncul, klik Python Environments, pilih interpreter dari (energi_hijau).

Artinya: Terminal dan kernel menyambungkan VSCode ke environment, memastikan kode menggunakan Python 3.12 dan library.

Sequence: Setelah setup VSCode, langkah ini menyambungkan ke environment untuk coding.

6. Menjalankan Jupyter Notebook

Narasi: Halo, teman-teman! Kita coba jalankan Jupyter Notebook di VSCode, seperti menyalakan mesin untuk analisis energi hijau. Ini langkah seru untuk melihat kode hidup!

Langkah:

  • Di analisis_energi_hijau.ipynb, klik + Code untuk membuat cell baru.
  • Tulis kode tes:
print("Halo, proyek Energi Hijau 2025 dimulai!")
  • Jalankan cell dengan tombol Run Cell (segitiga) atau Ctrl+Enter.
  • Lihat output di bawah cell, misalnya Halo, proyek Energi Hijau 2025 dimulai!. Jika muncul, setup sukses!
  • Tambah cell untuk query berikutnya, simpan file.

Artinya: Jupyter memungkinkan coding langkah demi langkah, ideal untuk analisis data.

Sequence: Setelah kernel siap, kita mulai coding di Jupyter, fondasi untuk query.
