Query 1: Cek Kepatuhan Pajak Karbon

Deskripsi

Tujuan: Mengecek apakah emisi perusahaan dari CSV melebihi batas pajak karbon (50 ton CO2).

Konsep: If-else (Chapter 5), Pandas untuk baca CSV.

Output: Status pajak (kena/bebas) untuk setiap perusahaan.

Kode

# Impor Pandas untuk mengelola data CSV
import pandas as pd

# Baca file CSV emisi_perusahaan.csv dari folder proyek
df = pd.read_csv('C:/EnergiHijau2025/emisi_perusahaan.csv')

# Definisikan batas pajak karbon (50 ton CO2, sesuai regulasi sederhana)
batas = 50

# Iterasi setiap baris di dataframe untuk cek emisi
for index, row in df.iterrows():
    # Ambil nilai emisi dari kolom Emisi_2024
    emisi = row['Emisi_2024']
    # Ambil nama perusahaan untuk output
    perusahaan = row['Nama_Perusahaan']
    # Cek apakah emisi melebihi batas menggunakan if-else
    if emisi > batas:
        # Cetak status kena pajak jika emisi > 50 ton
        print(f"{perusahaan} kena pajak karbon dengan emisi {emisi} ton!")
    else:
        # Cetak status bebas pajak jika emisi <= 50 ton
        print(f"{perusahaan} bebas pajak karbon dengan emisi {emisi} ton!")

Query: If-else dengan Pandas untuk cek emisi dari CSV.

Mengapa: Simulasi regulasi karbon sederhana, perkenalkan logika pengkondisian dan pengolahan data CSV.

Step-by-Step Penjelasan dan Artinya

  • Impor Pandas: import pandas as pd mengimpor library untuk baca CSV. Artinya: Siapkan alat untuk mengelola data tabel.

  • Set batas: batas = 50 definisikan batas pajak karbon. Artinya: Tetapkan standar regulasi untuk cek kepatuhan.
  • Iterasi dataframe: for index, row in df.iterrows() loop setiap baris. Artinya: Periksa emisi perusahaan satu per satu.
  • Ambil emisi: emisi = row['Emisi_2024'] ambil nilai emisi. Artinya: Dapatkan data emisi untuk cek pajak.
  • Ambil nama: perusahaan = row['Nama_Perusahaan'] ambil nama perusahaan. Artinya: Identifikasi perusahaan untuk output.
  • Logika if-else: if emisi > batas cek emisi, cetak status. Artinya: Tentukan kewajiban pajak berdasarkan regulasi.
  • Output pesan: print(...) tampilkan status pajak. Artinya: Beri informasi kepatuhan untuk laporan pemerintah.
